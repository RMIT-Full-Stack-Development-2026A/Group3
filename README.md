## Getting Started

This project is a **Monorepo** using NPM Workspaces. Follow these steps to run the entire system:

### 1. Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [MongoDB Atlas](https://www.mongodb.com/atlas/database) account (or local MongoDB)

### 2. Installation
From the root directory, run:
```bash
npm install
```
### 3. Running the Project
You need to run both the Backend and Frontend simultaneously. Open two terminal windows:

**Terminal 1 (Backend):**
```bash
npm run dev:backend
```

**Terminal 2 (Frontend):**
```bash
npm run dev:frontend
```

---

## Project Structure
- `packages/backend`: Node.js/Express server (API)
- `packages/frontend`: React/Vite application (Client)
- `packages/shared`: Shared logic (Game rules, AI logic)
